# SuperStore Sales Dashboard

## Project Overview

This project is an interactive Power BI dashboard created using the SuperStore Sales dataset. The dashboard helps analyze sales, profit, quantity, customer segments, regions, and product performance.

## Tools Used

* Power BI
* Microsoft Excel / CSV
* Microsoft PowerPoint
* GitHub

## KPIs

* Total Sales
* Total Profit
* Total Quantity
* Total Orders

## Dashboard Features

* Monthly Sales Trend
* Category-wise Sales Analysis
* Region-wise Profit Analysis
* Segment-wise Sales Analysis
* Top 10 Products
* Ship Mode Analysis
* Interactive Slicers

## Slicers Used

* Region
* Category
* Segment
* Year
* Ship Mode

## Key Insights

* Technology category generated the highest sales
* Consumer segment contributed the highest sales
* Central region produced the highest profit
* Standard Class was the most used shipping mode
* Top 10 products contributed a large portion of revenue

## Conclusion

The dashboard provides a clear overview of business performance and supports better decision-making through interactive visuals and filters.
